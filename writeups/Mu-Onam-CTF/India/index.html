Recon

The challenge had a lnk opening we get.

Searching through the web page we found a suspicious file in the contact section.

Dwnloading the file :

mykey.pub

By taking a look into the file we can understand that this is a RSA public key.

Finding the Flag

Fortunatley for us there is tool in github called RsaCtfTool

Cloning the github repo:

git clone https://github.com/Ganapati/RsaCtfTool.git
cd RsaCtfTool

Trying to crack the file using RsaCtfTool:

./RsaCtfTool.py --publickey mykey.pub --uncipherfile flag_abf168c3575c05707c5d88fa075758de.enc > out_flag
